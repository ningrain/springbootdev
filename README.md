# 安装
### 1、yum方式安装   
1. Docker 要求 CentOS 系统的内核版本高于 3.10 ，查看本页面的前提条件来验证你的CentOS 版本是否支持 Docker 。   
    - 查看系统内核版本：  
    `uname -r`   
2. 使用 root 权限登录 Centos。确保 yum 包更新到最新。   
    `yum update`   
3. 设置yum源  
    `yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo`
